# RVA Music Calendar

A simple, fast static site displaying live music events in Richmond, Virginia.

## Tech Stack

- **Astro** - Static site generator
- **Tailwind CSS** - Styling
- **Vercel** - Hosting & analytics

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Lint code
pnpm lint

# Build for production
pnpm build
```

## Features

- Embedded Google Calendar showing RVA music events
- One-click calendar URL copying with toast notifications
- Rubik font loaded via Google Fonts with performance optimizations
- Fully static site for optimal performance

## Deployment

Deploys automatically to Vercel when pushed to main branch.
